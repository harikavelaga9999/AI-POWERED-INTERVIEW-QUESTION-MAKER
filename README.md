# IT & ML Interview Questions Dataset

Welcome to my IT & ML Interview Questions Dataset repository!  

This repository contains a comprehensive collection of questions categorized by **IT and Machine Learning roles**, including Data Scientist, ML Engineer, MLOps Engineer, Data Engineer, and more. Each role includes:

- Sub-roles
- Key skills
- Tools & programming languages used
- Questions at beginner, intermediate, and advanced levels

- Covering 20+ IT & ML roles
